Regimes:
--- online - start online packing algorithm
--- pack - start offline packing algorithm

Online:
--- gen_time - the frequency of generation of the new virtual machine
--- limit_calls - number of generations
--- regime: (first or optimal)

Pack:
--- budget_migration - number of possible migrations

Servers:
--- count_type - number of server's types

    Parameters of server:
    --- core - number of cores
    --- memory - number of memory
    --- count - number of servers of this type

VirtMac:
--- count_type - number of virtual machine's types

    Parameters of server:
    --- core - number of cores
    --- memory - number of memory
    --- count - number of virtual machines of this type
